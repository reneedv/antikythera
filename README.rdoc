= antikythera
  A gem that adds some useful features to Time.
  
  {<img src="https://secure.travis-ci.org/reneedv/antikythera.png" />}[http://travis-ci.org/reneedv/antikythera]

== Install:
  $ gem install antikythera

== USAGE:
  * Time.now.before?(Time.now + 4)
    $ true
  * Time.now.after?(Time.now - 500)
    $ true
